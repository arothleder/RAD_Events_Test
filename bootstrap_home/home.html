FAQs

  • ● How do I earn points? Click "I Attended" button on My Events page.

  • ● Can I save my points for a RAD Event next quarter? Yes!

  • ● How many points are events worth? Events vary in points based on number of attendees, time spent, and attendee involevment.

  • ● What is the end-of-quarter event? The end of quarter event is a fun event that we ensure all users will want to go to. Attend the events on you 'My Events' page, and you can potentially go the end-of-quarter for FREE!

  • ● What if I do not have enough points for the end of quarter event? You can use your remaining points to get a discounted price on the end-of-quarter event ticket.
